# Apache JMeter

## Setup and Start JMeter

Download: apache-jmeter-5.5.tgz

```
$ tar xvzf apache-jmeter-5.5.tgz
$ mv apache-jmeter-5.5 ~/local/

$ cd apache-jmeter-5.5/
$ bin/jmeter
```
